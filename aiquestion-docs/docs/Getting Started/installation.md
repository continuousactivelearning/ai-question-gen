# Installation

## Prerequisites
- Node.js >= 16
- Python >= 3.10
- Poetry for dependency management
- Docker (optional)

## Clone repository
```
git clone https://github.com/kja8586/AIQuestionGeneration.git
```

## Backend Setup
```bash
cd backend
```
Using Poetry
```
poetry install
poetry shell
```
Using requirements.txt
```
pip install -r requirements.txt
```
